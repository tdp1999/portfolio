Selected Work — transition variants

Four ways to handle the panel swap when the active project changes. Each variant has its own tabs so you can click between projects and feel the motion. All variants assume the gallery is height-locked (aspect-ratio: 4 / 3) so image-count differences never shift the surrounding layout.

V1 — Crossfade whole panel

Outgoing panel fades out, incoming fades in. 360ms ease-out with a 60ms lead so the swap doesn't read as a flash. Opacity only — simplest treatment, cohesive but a touch flat.
